# Binder_RP_MVD
AutoHotkey скрипт для Roblox, созданный для упрощения ролевых отыгровок в МВД.  Включает 8 основных макросов на клавиши Numpad1-8: чистка ПМ, представление, ксива,  наручники/обыск, рация, Миранда, арест, ПМ в боевое положение. Есть настройки клавиши чата (Ctrl+Numpad4).  Использование на свой страх и риск!
